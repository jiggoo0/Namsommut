PRIVACY POLICY

DOCUMENT ID: NSM-P-2025 // LEVEL: TOP SECRET // ZERO-TRACE

1. นโยบายกำจัดร่องรอย (Zero-Trace Policy)

"นามสมมุติ" ถือว่าความลับของลูกค้าคือพันธกิจสูงสุด ข้อมูลที่ใช้ในปฏิบัติการทั้งหมดจะถูกจัดเก็บในรูปแบบ **Volatile Memory (RAM-Only)** ซึ่งไม่มีการบันทึกลง Hard Drive ถาวร ข้อมูลจะถูก Wipe (ทำลายทิ้ง) อย่างถาวรด้วยมาตรฐาน DoD 5220.22-M ทันทีหลังปิดภารกิจ

2. การระบุตัวตนและข้อมูลขั้นต่ำ

เรายึดหลักการจัดเก็บข้อมูลน้อยที่สุด (Data Minimization) เพื่อป้องกันความเสี่ยงแก่ผู้ใช้บริการ:

  • ไม่จำเป็นต้องใช้ชื่อจริง - สนับสนุนให้ใช้นามสมมุติในการสื่อสาร
  • ไม่เก็บ Log หมายเลข IP - ทุกการเข้าถึงเว็บนี้จะถูกปกปิดผ่านระบบ Secure Proxy
  • ข้อมูลเป้าหมาย - จะถูกใช้เพื่อการวิเคราะห์ใน "ห้องปฏิบัติการปิด" เท่านั้น

3. มาตรการความปลอดภัยทางไซเบอร์

ทุกการสื่อสารระหว่างคุณและ Agent ของเรา จะต้องผ่านช่องทาง **End-to-End Encryption (E2EE)** เท่านั้น เราขอสงวนสิทธิ์ไม่รับผิดชอบข้อมูลที่รั่วไหลจากการส่งผ่านช่องทางปกติ (เช่น SMS หรือ Email ธรรมดา) แนะนำให้ใช้ Telegram Secret Chat หรือ Signal เป็นหลัก

4. เส้นทางการเงินและธุรกรรม

เพื่อความปลอดภัยสูงสุดของลูกค้า เราสนับสนุนการชำระค่าปฏิบัติการผ่านสกุลเงินดิจิทัล (Cryptocurrency) เพื่อป้องกันการตรวจสอบเส้นทางการเงิน (Financial Trace) โดยเราจะจัดเตรียมกระเป๋าเงินชั่วคราว (One-time Wallet) ให้ในแต่ละเคส

AES-256 BIT ENCRYPTION ENABLED
